# Contributing to BNPL Dashboard

First of all, thank you for your interest in improving this project!

## How to Help
1. **Report Bugs**: Open an issue describing the bug and how to reproduce it.
2. **Feature Requests**: Shared your ideas through issues.
3. **Pull Requests**: 
   - Fork the repository.
   - Create a new branch (feature/amazing-feature).
   - Commit your changes with clear messages.
   - Open a PR for review.

## Code Standards
- Ensure your code follows the existing TypeScript/Next.js structure.
- Maintain the licensing headers at the top of your files.
