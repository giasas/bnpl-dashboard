# BNPL Dashboard

![License: GPLv3](https://img.shields.io/badge/License-GPLv3-blue.svg)
![Maintenance](https://img.shields.io/badge/Maintained%3F-yes-green.svg)
![Next.js](https://img.shields.io/badge/Next.js-14-black)

> Ένα προηγμένο Web3 interface για τη διαχείριση multi-collateral Buy Now Pay Later vaults, βασισμένο σε Next.js 14 και ethers.js.

## Features
- Multi-Collateral Vaults: Υποστήριξη πολλαπλών tokens ως εγγύηση.
- On-Chain Integration: Άμεση σύνδεση με το MultiCollateralBNPLVault smart contract.
- Premium UI: Σχεδιασμένο με Tailwind CSS & Shadcn UI για μέγιστη ταχύτητα και αισθητική.

## Installation & Usage
```bash
git clone https://github.com/giasas/bnpl-dashboard
cd bnpl-dashboard
npm install
npm run dev
```
